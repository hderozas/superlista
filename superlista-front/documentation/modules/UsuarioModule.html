Description

Módulo principal para las funcionalidades del usuario en la aplicación SúperLista. Agrupa todos los componentes relacionados con:

  • Gestión del hogar (home del usuario).
  • Ingredientes, recetas, menús y listas.
  • Creación y edición de menús. Este módulo está protegido por rutas con guardas (AuthGuard) y se carga bajo la ruta 'superlista'.
